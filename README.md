📌 Description

A console-based Student Management System built in C++ that allows teachers to manage student records and students to view their own information. The system includes authentication for teachers and stores data using file handling.

⚙️ Features
Teacher login authentication
Add and update student records
View all students (teacher access)
Student can view own data
File handling for data storage
🧠 Concepts Used

C++, File Handling, Arrays, Functions, Authentication, Menu-driven program
